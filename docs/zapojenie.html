Zapojenie - hasičské stopky - Arduino

Hardvér pre projekt:

  • Arduino Uno
  • LCD displej 16x2 alebo 20x4 s I2C prevodníkom
  • 4x spínacie kontakty
  • Možno doplniť o buzzer (hlásič), ktorý pípa pri ukončení útoku

    • Režim I2C zbernice vzhľadom na verzie stopiek

    Verzia stopiek Režim I2C
    Na desatiny Standard Mode 100 kHz
    Na stotiny Fast Mode 400 kHz
    Spotreba: 20mAh pri 5V napájaní
